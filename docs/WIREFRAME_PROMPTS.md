# SkillSwap Wireframe Builder Prompts

## ğŸ“‹ Application Overview

**Application Name**: SkillSwap  
**Type**: Peer-to-Peer Skill Exchange Platform  
**Platform**: Web Application (Next.js) with Mobile Companion  
**Primary Purpose**: Enable users to exchange skills through mutual teaching and learning  

### Core Concept
SkillSwap is a community-driven platform where users can:
- List skills they can teach (with detailed profiles, photos, availability)
- Browse and discover skills they want to learn
- Send exchange requests to skill providers
- Schedule and conduct learning sessions (online/in-person)
- Rate and review completed exchanges
- Build reputation through successful teaching/learning

### Color Palette
- **Primary**: Blue (#3B82F6) - Trust, learning, communication
- **Secondary**: Purple (#8B5CF6) - Creativity, innovation
- **Success**: Green (#10B981) - Completed exchanges, achievements
- **Warning**: Amber (#F59E0B) - Pending requests, alerts
- **Error**: Red (#EF4444) - Cancelled, rejected
- **Neutral**: Gray scale (#F9FAFB to #111827)

### Typography
- **Headings**: Inter font family, weights 600-700
- **Body**: Inter font family, weight 400-500
- **Captions**: Inter font family, weight 400, smaller size

### Visual Style
- **Design Approach**: Clean, modern, accessible
- **Layout**: Card-based with plenty of white space
- **Icons**: Lucide React icon library
- **Images**: Rounded corners, subtle shadows
- **Interactions**: Smooth transitions, hover states
